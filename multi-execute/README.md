## Execute Any Command on Mass

Generic shell scripts to execute a single command in multiple contexes

[execCommits.sh](https://github.com/MicahMaphet/dev-tools/blob/master/multi-execute/execEachCommit.sh) execute given command in commits given from stdin

[execDirs.sh](https://github.com/MicahMaphet/dev-tools/blob/master/multi-execute/execDirs.sh) execute given command in all of the subdirectories

[execEachCommit.sh](https://github.com/MicahMaphet/dev-tools/blob/master/multi-execute/execEachCommit.sh) execute given command in last n commits
